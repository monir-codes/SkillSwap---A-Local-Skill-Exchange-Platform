# SkillSwap - Professional Skill Development & Consultation Platform

**SkillSwap** is a premium learning and consultation platform where individuals can purchase expert-led sessions to master new skills. From professional coding to creative arts, the platform connects learners with experts through a seamless booking system, helping users invest in their personal and professional growth.

### ‚ú® Key Features
* **Secure Authentication:** Integrated with Firebase for robust Email/Password and Google Social Login.
* **Dynamic Profile Management:** Personalized user dashboard to view and update profile information (Name, Photo).
* **Session Booking System:** Interactive "Book Session" functionality with real-time slot tracking and pricing.
* **Password Recovery:** A dedicated "Forgot Password" flow that redirects users to their email for secure resets.
* **Protected Routes:** Secure 'My Profile' and 'Skill Details' pages to ensure data privacy for registered users.
* **Modern UI/UX:** Built with Tailwind CSS and DaisyUI, featuring smooth animations and a fully responsive design for all devices.
